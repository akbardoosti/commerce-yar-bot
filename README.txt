# RNOMPA Commerce Yar Bot

A WordPress plugin for integrating Telegram bot-based automation and notifications with WooCommerce or WordPress sites.

---

## Description

**RNOMPA Commerce Yar Bot** is a WordPress plugin that connects your WooCommerce or WordPress-powered store to Telegram. It allows administrators to receive automated updates, notifications, and manage commerce-related activities via a Telegram bot.

Key features include:
- WooCommerce order and product notifications via Telegram
- Admin and public hooks for extending plugin behavior
- Easy activation/deactivation and update mechanism
- REST API integration
- Admin settings and customizable options

---

## Installation

1. Upload the plugin folder (`commerce-yar-bot`) to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure the plugin via the settings page added to your WordPress admin dashboard.
4. Set up your Telegram bot token and connect it as instructed in the plugin settings.

---

## Usage

Once installed and configured, the plugin will automatically handle WooCommerce-related events and send notifications to your designated Telegram account or group. Advanced users can use available hooks to extend the plugin functionality.

---

## Frequently Asked Questions

**Q: How do I get my Telegram bot token?**  
A: Create a new bot via [@BotFather](https://t.me/BotFather) on Telegram and copy the token provided.

**Q: What events are supported for notification?**  
A: Order status changes, product updates, and custom WooCommerce events.
